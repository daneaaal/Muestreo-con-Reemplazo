## Contenido

### Muestreo con Reemplazo

Análisis estadístico descriptivo aplicado mediante **Muestreo Aleatorio Simple con Reemplazo (MASR)** sobre una población de 60 datos simulados y una muestra de 20 observaciones.

### Temas desarrollados

- Definición del muestreo con reemplazo  
- Procedimiento de selección aleatoria con Python  
- Población y muestra obtenida  
- Tabla de frecuencias absolutas, relativas y acumuladas  
- Medidas de tendencia central:
  - Media
  - Mediana
  - Moda

- Medidas de dispersión:
  - Varianza muestral
  - Desviación estándar
  - Rango

- Representación gráfica:
  - Histograma
  - Frecuencias acumuladas
  - Comparación población vs muestra

### Herramientas utilizadas

- Python  
- NumPy  
- Matplotlib  
- PowerPoint  
- PDF académico  

### Resultados principales

- Media muestral: **40.70**  
- Mediana: **33.00**  
- Moda: **35**  
- Desviación estándar: **30.72**  
- Rango: **91**
